# ryanfarahani1.github.io

Personal portfolio website for **Rayan Farahani** (Data Analyst & AI Engineer).  
Live site: `https://ryanfarahani1.github.io/`

## Highlights

- **Fast, single-page** portfolio (HTML/CSS/JS) deployed via GitHub Pages
- **Interactive demos section** linking to deployed AI projects (Hugging Face Spaces)
- **Responsive UI** with accessibility basics (skip link, ARIA labels)

## Tech stack

- **HTML**: `index.html`
- **CSS**: `style.css`
- **JavaScript**: `script.js`

## Local development

Because this is a static site, you can run it with any local web server.

### Option 1: Python

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

### Option 2: VS Code / Cursor

Use any “Live Server” extension, or just open `index.html` directly (some browser APIs may behave differently without a server).

## Deployment

This repo is intended to deploy from the **`main`** branch using **GitHub Pages**.
